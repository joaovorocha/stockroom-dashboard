UPS Chrome Extension

This extension enables Ship at UPS: one-click from Stockroom Dashboard to UPS CampusShip with auto-filled customer data.

Download Extension (.zip)

If download is blocked, try right-click → “Save link as…”. Alternate: .tar.gz

Install (Chrome)

  1. Download the zip above and unzip it into a folder (you should see a manifest.json file).
  2. Open chrome://extensions/
  3. Enable Developer mode (top right).
  4. Click Load unpacked.
  5. Select the folder you unzipped (the one that contains manifest.json).
  6. Pin it: puzzle icon → pin “SuitSupply Shipment Capture”.

How it works

  • On /shipments, review captured shipments (email import) and tracking status.
  • UPS CampusShip opens. A small SuitSupply widget appears on the page (bottom-right). Click Fill This UPS Form.
  • If the pending list is empty, open the dashboard in another tab (so the extension can sync shipments), then click Refresh in the widget.
  • The extension fills name, phone, email, address, service, weight, and references.
  • After label creation, the extension can send the tracking number back to the dashboard.
