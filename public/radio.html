How to use
1) Open this page and click Start (Manager only).
2) On the walkie, hold PTT 2–4 seconds while you speak a full sentence, then release.
3) Wait ~2 seconds after you stop talking (the system needs silence to finalize).
The Audio meter should move when there’s RF/voice coming in.
What do the buttons/settings mean?
Copy All: copies the entire transcript list to your clipboard.
Clear Screen: clears only your browser view (does not delete server logs).
Start: starts the transcription service on the server.
Stop: stops the transcription service on the server.
Kill RTL Lock: kills any stuck rtl_fm process that is holding the USB dongle (fixes usb_claim_interface error -6).
Frequency: radio channel frequency. For CLP446e channel 2: 446.01875M.
Gain: SDR gain. 0 means automatic (recommended).
VAD Threshold: how loud it must be to count as “speech”. Lower = more sensitive; higher = fewer false triggers.
Squelch: mutes noise. If you see “No PCM from rtl_fm” or the meter doesn’t move, set this to 0.
Squelch Delay: how long squelch stays open after it detects signal. Larger can help avoid chopping the first word.
PPM: frequency correction for the dongle. Usually 0 unless your dongle is off-frequency.
Save Settings: saves the values for next time.
Save + Restart: saves and restarts the service so changes apply immediately.
