# ups-api-shipmentsv1 (snapshot)

This folder is a frozen snapshot of the previous UPS shipments implementation (UI + API logic) before the app was simplified to a read-only “shipment capture visualization” view.

Included (original paths):
- `public/shipments.html`
- `public/shipments-processing.html`
- `routes/shipments.js`
- `utils/ups-email-parser.js`
- `utils/upsApi.js`

Note: These files are not automatically served by the app from this folder; they exist as a restore reference.
